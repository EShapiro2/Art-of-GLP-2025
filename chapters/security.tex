This chapter presents the security foundations of \GLP.

\section{Cryptographic Primitives}

\GLP integrates standard cryptographic primitives:
\begin{itemize}
\item Public-key cryptography for identity
\item Symmetric encryption for message privacy
\item Digital signatures for authenticity
\item Hash functions for integrity
\end{itemize}

\section{Blockchain Security Properties}

\GLP streams enjoy blockchain security properties:

\begin{formal}{Stream Security}
\begin{fprop}[Stream Immutability]
Once a message is added to a stream, it cannot be modified or removed without detection.
\end{fprop}

\begin{fprop}[Stream Ordering]
Messages in a stream have a total order agreed upon by all readers.
\end{fprop}
\end{formal}
